                             QUICK START
================================================================================

1. CREATE MONGODB CLUSTER (2 minutes)
   → Go to mongodb.com/cloud/atlas
   → Create free cluster
   → Create database user
   → Get connection string

2. SET UP ENVIRONMENT (1 minute)
   → Create .env.local file
   → Add MONGODB_URI
   → Add JWT_SECRET

3. INSTALL & RUN (2 minutes)
   → npm install
   → npm run dev
   → Open http://localhost:3000

4. TEST PLATFORM (5 minutes)
   → Sign up as creator (5 steps)
   → Sign up as brand
   → Create campaign
   → Apply to campaign
   → Verify data in MongoDB
